AiPhlo Navigation Systems

Hosted assets for Webflow integration.

Files

  • css/aiphlo-nav.css - All navigation styles
  • js/aiphlo-nav.js - All navigation scripts

Usage in Webflow

Add these to your Webflow site via Custom Code or HTML Embed:

CSS (Head Code)

<link rel="stylesheet" href="https://[your-username].github.io/aiphlo-webflow/css/aiphlo-nav.css">

JavaScript (Footer Code)

<script src="https://[your-username].github.io/aiphlo-webflow/js/aiphlo-nav.js"></script>
